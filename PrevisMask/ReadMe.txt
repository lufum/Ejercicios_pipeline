
Install instructions

1)place the file inside plug-ins directory into C:\Users\userName\Documents\maya\version\plug-ins
2)place the files inside scripts directory into C:\Users\userName\Documents\maya\version\scripts
3) Open Autodesk Maya
4) Run the next lines in the script editor

import previsMask
previs= previsMask.previsMask()

optional
5)to create a shelf button, in the script editor click file->save script to shelf
